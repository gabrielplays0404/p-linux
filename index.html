Sobre a Paper Linux

Unindo a estabilidade inabalável do Debian com a leveza extrema do IceWM.

Notas de Lançamento v1.0

  • Base Debian Stable (2026)
  • Correção de boot 32-bit (nomodeset)
  • Kernel otimizado para baixo consumo de RAM

Requisitos

512MB RAM Mínimo

CPU 1GHz

2GB Espaço

Interface do Sistema

Ambiente IceWM

Terminal Leve

Como Instalar

Abra o terminal (Xterm) e digite o comando abaixo:

sudo paper-install
